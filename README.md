# 프로젝트1 : 파이값 구하기
파이값을 구하고 그래프로 그림
 - <img src = "img/파이수식.png">
```
p=1
pilist = []
for n in range(1,1000):
    p = p * ((2 * n + 1) ** 2 - 1)/(2 * n + 1) ** 2
    # print(p*4, ',')
    pilist.append(p*4)

import matplotlib.pyplot as plt
plt.plot(pilist)
plt.show()
```
